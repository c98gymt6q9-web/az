# 🎯 Telegram Sales Bot

Полнофункциональный Telegram-бот для продажи файлов с интеграцией Google Drive.

## ✨ Основной функционал

- 💳 **Приём платежей** - показываем твои реквизиты (Яндекс.Касса, Bitcoin, крипто и т.д.)
- 📧 **Проверка Gmail** - приём только @gmail.com адресов
- 📂 **Автоматический доступ** - добавляем пользователя в Google папку после покупки
- 💾 **База данных** - ведём логи всех покупок (SQLite)
- 📊 **Admin панель** - просмотр статистики и аналитики
- 🔒 **Безопасность** - использование Service Account для Google Drive

## 🚀 Быстрый старт

### Для спешащих (2 минуты):
1. Получи токен от [@BotFather](https://t.me/botfather)
2. Создай папку в Google Drive и скопируй ID
3. Создай Service Account в Google Cloud Console
4. Заполни конфиг в `main.py`
5. `python main.py`

**Полная инструкция:** см. [QUICKSTART.md](./QUICKSTART.md)

## 📋 Требования

- Python 3.9+
- Telegram аккаунт
- Google аккаунт (для Drive и Cloud Console)
- Интернет подключение

## 💻 Установка

```bash
# 1. Установить зависимости
pip install -r requirements.txt

# 2. Скопировать service_account.json в папку проекта

# 3. Заполнить конфиг в main.py

# 4. Запустить бот
python main.py
```

## ⚙️ Конфигурация

### Минимальная конфиг (в main.py):


## 🎯 Как работает

### Процесс покупки:

```
1. Пользователь пишет /start
   ↓
2. Нажимает "💳 Купить продукт"
   ↓
3. Вводит свой Gmail
   ↓
4. Видит твои реквизиты (номер счета, контакты и т.д.)
   ↓
5. Переводит деньги
   ↓
6. Нажимает "✅ Я оплатил(а)"
   ↓
7. Бот проверяет что это Gmail
   ↓
8. Бот добавляет юзера в Google папку
   ↓
9. Юзер получает доступ к файлам ✅
```

## 🔒 Безопасность

⚠️ **Важно!**

- ✅ Никогда не коммитьте в git:
  - `service_account.json`
  - TELEGRAM_TOKEN
  - `.env`

- ✅ Используйте переменные окружения (.env файл)

- ✅ Проверяйте доступы в Google Cloud регулярно

Пример с .env:
```python
from os import getenv
TELEGRAM_TOKEN = getenv("TELEGRAM_TOKEN")
```

## 🆘 Решение проблем

### Бот не запускается?
- Проверь TELEGRAM_TOKEN
- Убедись что Python 3.9+
- Посмотри ошибку в консоли

### Не добавляет в папку?
- Проверь email это @gmail.com
- Убедись Service Account имеет доступ
- Проверь GOOGLE_FOLDER_ID
